# 0xfinder's dotfiles

Repository for my personal dotfiles, yes. 

## Packages

- Kitty (Shell)
- Git (VCS)
- Htop (Process viewing)
- Mpd (Music daemon)
- Ncmpcpp (Frontend for Mpd)
- Picom (Window Compositor)
- Latte Dock (For status bars)
- Neovim (Editor)
- Xorg (X settings)
- VSCode (IDE)
- KDE Plasma (DE)
- Neofetch (Fetcher)
